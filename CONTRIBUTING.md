# Contributing

Thanks for considering a contribution to YouTripio.

## Development Setup

1. Install dependencies: `npm install`
2. Run the dev server: `npm run dev:3022`
3. Open the app in your browser at `http://localhost:3022`

## Branching

- Use feature branches: `feature/<short-name>`
- Keep commits focused and descriptive.

## Code Style

- TypeScript + App Router conventions.
- Tailwind classes should be readable and consistent.
- Prefer small components and reuse UI primitives.

## Pull Requests

- Clearly describe what changed and why.
- Include screenshots for UI changes.
- Ensure `npm run lint` passes.
